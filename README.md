# Unchain Admin

A vue.js2 fullstack project with Cloudflare Workers Functions



## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
